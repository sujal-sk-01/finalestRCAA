# RCAAgent-Env

OpenEnv environment simulating SRE incident response at hyperscale. AI agents investigate failing microservices using metrics, traces, and logs to identify root causes and write RCA reports. Three difficulty levels with deterministic grading. Built for the Meta x Hugging Face OpenEnv Hackathon.
